CARDS_RULE
{
Les règles sont simples. Blackjack. 36 cartes, des six aux as. L’as vaut 11 points, le roi 4, le Lord (cavalier) 3, le Nobile (fantassin) 2, les autres valent leur valeur indiquée. Le but est d’atteindre 21 points.
Cliquez sur le paquet pour tirer une carte de plus. Chaque carte est une mise. Cliquez sur le portrait en haut à droite pour terminer votre tour. Il vous faut assez d’argent pour au moins trois mises pour jouer. Appuyez sur Échap ou sur la croix pour quitter.
}
DICE_RULE {Craps. Cinq dés pour chaque joueur. Vous pouvez relancer tous les dés ou seulement certains, mais une seule fois. Chaque dé misé compte comme une mise. Votre objectif est d’obtenir le score le plus élevé.\nVoici les combinaisons : duo (2), deux duos (2+2), triade (3), full (3+2), carré (4), suite (dés consécutifs) et poker (5).\nCliquez sur un gobelet pour lancer, sur un dé pour relancer ce dé précis, sur un portrait pour terminer votre tour (si vous ne souhaitez pas relancer ou n’avez pas assez d’argent).}
