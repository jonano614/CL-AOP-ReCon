SeaBattle_title {Batalla naval}
SeaBattle_descr
{
¡Bienvenido a la batalla naval, Capitán!
Tu objetivo principal: sobrevivir y hundir los barcos enemigos.
~
Cada barco tiene tres estadísticas principales: casco, velas y tripulación. Si el casco llega a cero, el barco se hunde. Las velas dañadas te ralentizan, y perder tripulación facilita el abordaje y ralentiza todas las acciones del barco.
~
Cada estadística tiene su munición ideal: metralla para la tripulación, cadenas para las velas y mástiles, bombas para el casco. Las balas de cañón son versátiles y tienen el mayor alcance. El daño depende tanto del tipo de munición como del cañón que la dispara: cuanto más grande el cañón, mayor el daño.
~
Para disparar a un enemigo, usa el botón izquierdo del ratón cuando un barco enemigo esté dentro de tu arco de fuego.
~
Si la batalla te parece demasiado lenta, puedes acelerarla usando las teclas de control de tiempo ('+' y '-').
}
