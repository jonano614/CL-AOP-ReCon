“狂野加勒比”
这个免费特质——“狂野加勒比”——会让一些“普通”的随机遭遇变得疯狂且不可思议！如果你不想让你的冒险变得有点疯狂，请不要选择这个特质。请注意，选择后无法关闭！

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
“狂野加勒比”包含：
- 骷髅球体的出现；
- 打开宝箱时会出现骷髅，就像在洞穴中一样；
- 可接取“幽灵船”任务；
- 在“教堂需要帮助”任务中清除的是亡灵，而不是印第安人；
- 部分任务的视觉效果会发生变化。
}
InfoMessage_Text_1
{
新增冲刺机制——你可以通过冲刺来躲避伤害：
- 正确时机下，躲避手枪射击或挥砍的成功率为100%；
- 冲刺消耗的基础能量从3点降为2点，方向不限；
- 冲刺所需能量取决于你的装备重量（2点基础能量 + 0.1×装备磅数）；
- 能量不足时，冲刺成功率降低——精疲力竭的角色仍可冲刺，但躲避伤害的几率减半；
- 所有强力战士、老兵和军官有25%几率使用冲刺；
- 普通敌人不会冲刺。
}
InfoMessage_Text_2
{
全新弹道系统与船员伤害机制

船只与要塞的弹道系统已重做：
当敌人距离较近时，炮弹将以直线射击，因此海浪可能会为你或目标提供掩护（除非射击方高于浪头，比如某些要塞）。

所有火炮射击的散布范围增加，目的如下：
1. 提高小型船只对抗高级船只时的生存能力——现在它们在机动时更难被击中。
2. 强化优秀炮手官员（高精准与火炮属性）的价值。
3. 优化海战体验，避免高级船只仅凭几轮齐射就结束战斗。

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
船员伤害机制更新：现在必须先对船体造成伤害、降低其耐久度，才能对船员造成显著伤害。此机制同样适用于玩家的船只。

现在霰弹对船员的伤害将根据船体耐久百分比计算。详细说明可在日志簿的百科全书标签（船只海战）中查阅。
详细信息可在“文档”-“参考手册”-“船只海战”标签中找到。
}

InfoMessage_Text_4
{
快速搜刮
\n\n
如果被登船的船只的船长投降，战利品小队一定会进入他的船舱，搜寻所有隐藏物品，口袋也不会放过。
\n\n
如果你选择将所有战利品转移到船舱的箱子里，别忘了时常去检查。频繁登船后，水手们会把找到的一切都塞进去，箱子很快就会变得杂乱。
\n\n
如果船长或水手长拥有“钻石之眼”特质，船员会更加细心挑剔，完全无视垃圾，只收集更有用的东西。
\n\n
在海上，这种“挑剔”的效果取决于你水手长的“登船”技能：
- 7级和6级船无要求
- 5级船需15至29点
- 4级船需30至44点
- 3级船需45至59点
- 2级船需60至74点
- 1级船需75点以上
- 在陆地上无要求
\n\n
如果水手长和/或船员缺席，此选项不可用。
}
