Strength_hint
{
力量：让你攻击更有力，能承受伤害，也能携带更多物品。
是需要大量体力的技能（如“重型武器”、“中型武器”和“火炮”）的关键属性。
~
影响初始生命值和最大携带重量。
}
Perception_hint
{
洞察力：能够看见、听见、嗅出并察觉异常之事的能力。
决定以下技能的属性：“精准度”和“潜行”。
~
影响船只的航行距离及全球地图上的互动。
}
Agility_hint
{
反应：动作协调与手部灵巧。
是“轻型武器”“中型武器”“火器”以及“登船”等技能的关键属性。
~
影响最大精力值。
}
Charisma_hint
{
魅力：外表与口才的结合。高魅力对希望用言语影响他人的角色非常重要。
是“领导力”和“商业”等技能的关键属性。
~
决定你能雇佣的军官数量（A*2）。
}
Intellect_hint
{
天赋：知识、智慧，以及迅速做出正确决策的能力。

决定以下技能的主要属性：“商业”、“火炮”、“航海”。
~
影响角色升级和获得特质的速度。
}
Endurance_hint
{
耐力：生存能力、持久力和身体抗性。
是“修理”和“防御”等技能的关键属性。
~
每提升一级可增加生命值和负重上限。
}
Luck_hint

是“火器”、“运气”和“潜行”等技能的关键属性。
~
影响各种游戏事件及战利品数量。
}

Sailing_hint
{
能够在风暴和浓雾中安全将船只驶入港口，善于使用导航设备并掌控舰队。即使是最大最可怕的船只，在经验丰富的领航员稳健的掌控下，也能如飞鸟般轻盈前行。
~

~
缺乏航海技能会对所有技能和属性造成惩罚。

2级需要80
3级需要65
4级需要45
5级需要25
6级需要1

Rank_hint
{
等级表示角色在游戏中的成长进度，会随着角色的经验提升而提升。
~
随着角色等级的提升，一些属性也可能随之增加。
~
随着等级提升，角色会更频繁地遇到强大的对手，并获得更好的物品。
}
Life_hint
{
生命值表示角色在被击败或杀死前能承受的伤害量。也称为生命点（HP）。
~
每提升一个等级，生命值都会根据其耐力属性增加。
}
Health_hint
{
活力代表角色的整体身体状况：所有属性越差，活力也会越差。当角色受到多次伤害或频繁饮用劣质药剂时，活力会下降；若不参与近战，活力则会恢复。如果活力低于“极佳”，角色的技能和属性会受到惩罚。
~
活力可分为以下几个等级：
极差 - 较差 - 一般 - 中等 - 良好 - 极佳
~
指示器为金色时，表示当前状态已达到该角色的最大值，无法自行提升。为白色时，说明活力状态还可以恢复。
~
活力会自行缓慢恢复，尤其是在世界地图航行时更慢。情爱活动能最快恢复活力，酒馆睡觉次之。“极佳活力”能力可使恢复速度加倍。据说，饮用稀有的专用药剂也能提升活力。
~
教堂的神父可以恢复活力上限，但次数有限，幅度不大且价格昂贵。有时需要拜访2-3位神父，才能让“金色”变回“白色”，再次获得提升空间。但之后仍需等待活力自行恢复到极佳（“金色”）状态。
}
Energy_hint
{
能量表示角色的体力储备。
~
近战中的所有攻击动作都会消耗能量；战斗时角色会感到疲劳，需要休息来恢复体力。
~
反应（R）决定你的能量上限。
}
Money_hint {金钱才是万物之王。最常见的是西班牙比阿斯特，也许其中还夹杂着罕见的法国埃居、英格兰克朗、多面塔勒……又有什么区别？这些多样的银币，都是无国界的真银！}
Reputation_hint
{
声望是衡量社会如何看待角色的指标。
~
声望可以是高（正面）、中立或低（负面）。声望会根据角色的行为变化，并决定游戏中其他角色如何看待你。
~
普通角色的声望在整个游戏过程中保持不变：他们的性格已定，行为也较为一致。
~
主角的名声会随时间消退，他的功绩或失败会被人们渐渐遗忘。恶行比善举更难被忘记。如果主角既不选边，也没有重大作为，最终声望会变为“普通水手”。
}
weight_hint
{
角色在不降低移动速度和不受其他技能惩罚的情况下可携带的最大重量。超载时，角色移动极慢并会受到惩罚。
~
受力量和耐力影响。
}
Title_hint {头衔——一种荣誉称号，既代表晋升后的权威等级，也体现角色为其所效力帝国所做的贡献。}
NextExp_hint
{
经验阈值表示当前技能、等级和能力已分配的点数，以及获得下一个点数所需的经验值。
~
技能会在你执行相关动作时提升。例如，每次精准的手枪射击都会为“火器”技能获得经验点。这些技能点会计入等级经验和个人/船只能力（取决于技能类型）。
~
当等级经验达到阈值时，英雄将提升等级，并获得更多生命值和能力点，玩家可以自由分配。
}
TABLE_PERKS {角色能力}
TABLE_PERKS_desc
{
特质是角色的额外能力，可以带来新优势或解锁原本无法使用的游戏功能。通过特质，角色可以专精某一领域，也可以成为涉猎广泛的多面手。

~

双击所选特质可查看其描述及前置条件（如有）。

~

某些特质需要先解锁树状结构中的前置能力，无法立即学习。

Nation_hint
{
主角的国籍是在游戏开始时选择的国家，或当前所持通行证所属的国家（即主角效力的国家）。主角头像会显示对应国籍的旗帜。
~
你可以在“角色”界面查看国籍。在“国籍”界面可以看到你当前船只悬挂的旗帜——只能在海上更换，用于隐藏真实身份。请注意，公海上的船只和港口守卫可能会识破你的伪装。主角的潜行属性会影响伪装成功的几率。
~
敌对识别会根据各国之间的官方关系以及悬赏令来判定。如果主角被本国悬赏，即使对本国也会被视为敌人。但敌对国家在好感度高时可能会网开一面。
~
在敌对城市无法使用快速移动。
}

~
大多数情况下，你可以为他索要赎金，如果他是海盗，还可以招募他加入主角的船员。如果你运送的乘客因旅途太长而抱怨，也可以将其囚禁。船长则是那些在登船战中向你投降，或在海上被你救起的人。赎金最多通常由与俘虏同国籍的总督支付，但金额会波动，总督对俘虏价值的看法也会有所不同。
