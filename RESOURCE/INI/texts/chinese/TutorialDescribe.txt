SeaBattle_title {海战}
SeaBattle_descr
{
欢迎来到海战，船长！
你的主要目标：生存并击沉敌舰。
~
每艘船有三项主要属性：船体、帆和船员。船体归零时，船只沉没。帆受损会降低速度，失去船员则更容易被登船并减缓所有操作。

~
当敌舰进入射击扇区时，点击鼠标左键即可开火。
~
如果战斗节奏太慢，你可以用时间控制键（‘+’和‘-’）来加快速度。

BloodPrologue_SeaBattle_title {海战}
BloodPrologue_SeaBattle_descr
{
每艘船都有三项主要属性：船体、帆和船员。如果船体归零，船只就会沉没。帆受损会降低你的速度。
~
注意你的船员数量指示器。你的人手不足——操作帆的效率会大幅下降。目前所有资源都被分配给炮组。
~
每项属性都有对应的最佳弹药：霰弹用于打击船员，链弹用于破坏帆和桅杆，炸弹用于摧毁船体。实心弹则是通用弹药，射程最远。
~
要向敌舰开火，当敌舰进入你的射击扇区时，点击鼠标左键。
~
如果战斗节奏太慢，你可以用时间控制键（‘+’和‘-’）来加快速度。
}

BloodPrologue_SeaEscape_title {脱离战斗}
BloodPrologue_SeaEscape_descr
{
看起来西班牙人正在撤退。他们无法承受这样的火力！
~
通过拉开与敌舰的距离脱离战斗。你可以通过望远镜查看与敌舰的距离。
~
要使用望远镜，先进入甲板（默认按‘Tab’），然后按住‘Ctrl’键（默认）。
~
当与敌舰的距离达到1000时，你就可以安全离开战场。
}

BloodPrologue_Navigation_title {航海}
BloodPrologue_Navigation_descr
{
由于航海技能不足，你当前正受到属性惩罚！
~
航海主要决定角色能指挥多大等级的船只。
~
例如，5级船需要25点，4级船需要40点。你可以在角色界面右键点击技能图标查看各等级的要求。
~
要消除这些惩罚，你需要一位经验丰富的领航员。但在游戏初期几乎无法找到这样的人。
~
唯一的解决办法是将当前船只停靠，改为指挥低一级的船。
}
