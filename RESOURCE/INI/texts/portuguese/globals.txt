CARDS_RULE
{
As regras são simples. Blackjack. 36 cartas, de seis a ás. Ás vale 11 pontos, reis - 4, Lord (cavaleiro a cavalo) - 3, Nobile (soldado a pé) - 2, o restante vale o número indicado. Seu objetivo é fazer 21 pontos.
Clique no baralho para pegar mais uma carta. Cada carta é uma aposta. Clique no retrato no canto superior direito para encerrar sua vez. Você precisa de dinheiro suficiente para pelo menos três apostas para jogar. Pressione Esc ou o X para sair.
}
DICE_RULE
{
Craps. Cinco dados para cada jogador. Você pode rerrolar todos os cinco ou qualquer um deles, mas apenas uma vez. Cada dado é uma aposta. Seu objetivo é fazer a maior pontuação possível.
As combinações são: dupla (2), duas duplas (2+2), trinca (3), full (3+2), quadra (4), sequência (dados em ordem) e poker (5).
Clique no copo para lançar, clique em um dado para rerrolar aquele dado específico, clique no retrato para encerrar sua vez (caso não queira rerrolar ou não tenha dinheiro suficiente para isso).
}
